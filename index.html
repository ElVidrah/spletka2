Pogosta Vprašanja

  • Zakaj se to učim?

    • Kratek odgovor: Ker je to uporabno.
    Dolg odgovor: Kot del osnovnega znanja informatike je znanje osnov HTML in CSS nepogrešljiv. Te osnove vam omogočajo postavitev osnovne spletne strani za bodoče potrebe. Ta snov te ne bo naučila vsega o teh spletnih tehnologijah, ampak le osnove ter osnove koncepte ki ti bodo služile kot odskočna deska za nadaljni študi teh tehnologij.

  • Kako lahko pišem HTML in CSS

    • Kodo lahko pišeš v vseh osnovnih tekstovnih urejevalnikih kot so textpad, notepad++ in podobno. Obstajajo pa tudi bolj napredni urejevalniki besedil, ki omogočajo veliko uporabnih bljižnic, okrajšav ter barvne sintakse, ki močno olajšajo delo primer takega je Sublime Text.

  • NEDELA NEDELA NEDELAAAA!

    • Brez panike HTML sintaksa izgleda na prvi pogled zelo kompleksno vendar z malo vaje postane zelo pregledna in nazorna. Največ napak se pojavi prav pri pisanju različnih selektorjev ter elementov, saj tako HTML in CSS zahtevata korektno sintaksto.

      Nekaj pomoči pri pri reševanju napak:

    • Preveri sintakso, si pravilno napisal odpiralne in zapiralne značke?
    • Pri povezanih značkah kot so img, link, a preveri ali je pot do naslovljenega elementa pravila in ali je celotna (s tem mislim končnico danega dokumenta )
    • CSS probleme se najlažje rešuje z preverjanjem delovanja posameznih selektorjev. To storiš najlažje tako da dodaš prvilo za spremembo barve kot so color in background-color
    • Pri samih pravilih CSS poglej sintakso, nato pa poglej pravilo (ali obstaja ter kakšne lastnosti sprejme). Če ti ne gre je priporočljivo ustvariti posebno tatoteko HTML in CSS kjer lahko preizkušaš delovanje posameznih pravil in selektorjev.
    • Lahko si tudi pomagas z pregledovalnikom HTML in CSS v posameznih brskalnikih.
    • Kot vedno je najboljša in najhitrejša rešitev pri reševanju problemov Google. Z pravilnio uporabo besed lahko dokaj hitro najdes pravilno sintakso ter tako rešiš svojo napako.
    • Kateri spletni brskalnik uporabljaš, starejši še ne podpirajo nekatira novejša CSS pravila pri tem je Google tvoj najboljši prijatel.
    • Če vsi našteti nasveti ne obrodijo sadov me lahko vedno kontaktiraš, Če imaš srečo ti lahko celo mogoče pomagam.
